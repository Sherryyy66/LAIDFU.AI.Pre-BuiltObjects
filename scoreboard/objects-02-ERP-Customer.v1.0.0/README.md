## 版本与更改日志
- **v1.0.0** (2025-04-10): 初始版本，支持scorboard动态读取客户的部分指标信息。

## 功能概述
- 本示例(Objects)支持scorboard动态读取客户的部分指标信息。在[客户]自定义布局添加scoreboard，每次打开客户单据后都会自动读取当前客户的以下指标，方便用户快速获取部分信息。
- 已清缴应收账款金额
- 未清缴应收账款金额
- 已收订金
- 未收订金
- 销售订单总数
- 送货单总数
- 销售总额（最近一年）
- 销售总额（最近一月）
- 未交货订单数
- 退货订单数

## 产品功能要求
使用本Objects至少需要购买aiM18并安装以下模块功能。
-免费SaaS云应用设置

## 安装与配置步骤
1. 从本文件夹根目录下载安装包压缩文件“install_file.zip”。
2. 将压缩文件解压至本地文件夹。
3. 在 aiM18 中导入 Objects 文件和 Excel 文件。
4. 在 [客户] 自定义布局添加scoreboard。
